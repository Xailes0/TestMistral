# 🍽️ Gestion des Repas

*Ce fichier centralise tous les repas que je prends. Format simple pour un suivi facile et une mise à jour automatique.*

---

## 📅 **Format des Entrées**

Chaque repas est noté sous la forme :
```markdown
### [Date] - [Type de repas]
- **Aliments** : Liste des aliments/boissons consommés
- **Heure** : HH:MM (optionnel)
- **Lieu** : Où le repas a été pris (optionnel)
- **Notes** : Remarques (ex: "très bon", "trop salé", etc.) (optionnel)
```

**Exemple :**
```markdown
### 2024-10-15 - Petit-déjeuner
- **Aliments** : Café, Tartine (pain complet, beurre), Jus d'orange
- **Heure** : 08:30
- **Lieu** : Maison
- **Notes** : Bon équilibre
```

## 🔍 **Catégories et Tags**

Pour faciliter les recherches, utilisez des **tags** ou **catégories** si nécessaire. Exemples :
- `#vegetarien`
- `#sans-gluten`
- `#rapide`
- `#fait-maison`
- `#restaurant`

**Exemple avec tags :**
```markdown
### 2024-10-16 - Déjeuner
- **Aliments** : Salade de quinoa, Avocat, Tomates, Vinaigrette maison
- **Heure** : 13:00
- **Lieu** : Maison
- **Notes** : #vegetarien #fait-maison
```

## 🛠️ **Instructions pour Ajouter un Repas**

1. **Copiez** le format ci-dessus.
2. **Collez** à la fin de la section **Liste des Repas**.
3. **Remplissez** les informations.
4. **Sauvegardez** le fichier.

*Je peux ajouter des repas automatiquement si tu me donnes les informations sous forme de texte ou de liste.*

## 📌 **Notes Générales**

- Les repas sont classés par **date** (du plus récent au plus ancien ou inversement).
- Utilisez des **listes à puces** pour les aliments.
- Les champs **Heure**, **Lieu**, et **Notes** sont optionnels.
- Pour les **statistiques**, utilisez des outils externes ou des calculs manuels si nécessaire.
